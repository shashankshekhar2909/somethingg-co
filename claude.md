## WEBSITE PURPOSE
This website represents Somethingg as a modern IT consultancy and services company.

The website must:
- build trust quickly
- look premium and clean
- clearly explain services
- introduce the core team
- generate leads through contact / consultation inquiries

This is NOT a fashion brand website.
This is NOT a playful lifestyle e-commerce site.
This is a professional services website with a modern, human feel.

- minimal

## ANIMATION & 3D TECH STACK

### Libraries in use
- **Three.js** — 3D scene rendering for hero background (particle network, geometric shapes)
- **Anime.js** — entrance animations for hero text and UI elements

### Hero scene guidelines
- Three.js canvas: full-viewport, absolute positioned behind content
- Particles: floating network/constellation effect, electric blue palette
- Mouse parallax: subtle scene shift on mouse move
- Wireframe geometric shape in background (icosahedron or similar)
- Keep performance in mind — use instanced geometry where possible

### Animation principles
- All hero text animates in on mount (staggered, smooth)
- Anime.js for 2D UI animations (fade, slide, scale)
- Three.js for 3D background only
- No jarring or flashy effects — subtle, premium
